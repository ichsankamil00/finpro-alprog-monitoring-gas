1. Mengimplementasikan komunikasi client-server berbasis socket.
2. Membuat/memperbaiki `client.cpp` sebagai simulasi dummy ESP32.
3. Membuat/memperbaiki `server.cpp` sebagai penerima data sensor.
4. Mengatur format data final berupa `sensor_id`, `temperature`, `co_value`, `lpg_value`, `timestamp`, dan `status`.

6. Menyimpan seluruh data monitoring ke `data_semua.txt`.
7. Menyimpan data berstatus BAHAYA ke `data_kritis.json`.
8. Membantu setup GitHub repository dan working version program.
